## 🚀 Getting Started

### Prerequisites
```
✓ Node.js 
✓ npm/yarn
✓ Basic React knowledge
```

### Quick Start Guide

1. **Clone Repository**
   ```bash
   git clone https://github.com/yourusername/dr-cloud.git
   cd dr-cloud
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Launch Development Server**
   ```bash
   npm run dev
   ```
